# odin-recipes

This is a project where you code a basic recipes website using the beginner skills with HTML learned in the foundations portion of the Odin Project

Skills to be used:
 - Using proper file structure for an HTML project
 - Using relative and absolute links
 - Using ordered and unordered lists
 - Using images
 - Using headings effectively
